## 📖 Overview

Ghost Protocol is an immersive network-topology puzzle game developed for **Mosaic TechFest 2024** under **CSI SFIT**.

The application combines a nostalgic Windows 95-inspired user interface with an interactive graph-building challenge where participants must recreate a target network architecture before time runs out.

---

### 🔐 Secure Challenge Access
- Firebase Firestore authentication

## 🎮 Gameplay

1. Login using the challenge password.
2. Enter the Ghost Protocol environment.
3. Drag network components into position.
4. Create valid connections between nodes.
5. Reconstruct the target network architecture.
6. Complete the puzzle before the timer expires.
